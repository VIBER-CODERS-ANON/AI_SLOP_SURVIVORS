TILESET DIRECTORY
=================

This directory contains tileset images for the dungeon system.

brutal_dungeon_tileset.png:
- 128x128px image (4x4 grid of 32x32 tiles)
- Wang tileset for smooth transitions between stone and blood
- Generated by PixelLab MCP

The tileset uses corner-based transitions where each tile index (0-15) represents
a specific corner configuration for smooth blending between terrain types.
